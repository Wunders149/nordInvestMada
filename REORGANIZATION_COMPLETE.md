## 🚀 How to Use

### 1. Install Dependencies
```bash
npm install
```

### 2. Add Your Images
Place images in the appropriate folders:
```
/public/images/hero/main-hero.jpg                    # 1920x1080px
/public/images/projects/project-1-ramena.jpg        # 1200x675px
/public/images/team/about-main.jpg                  # 800x1000px
/public/images/services/icon-construction.svg       # 200x200px
```

### 3. Update HTML Image Paths
Open `public/index.html` and replace emoji placeholders with real images:

**Before:**
```html
<div style="font-size:8rem;opacity:0.08;">🏗️</div>
```

**After:**
```html
<img src="/images/hero/main-hero.jpg" alt="Construction">
```

### 4. Start Development Server
```bash
npm run dev
# Server runs on http://localhost:3000
```

## 🔧 Configuration Files Updated

### `package.json`
```json
{
  "main": "src/server.js",
  "scripts": {
    "start": "node src/server.js",
    "dev": "nodemon src/server.js"
  }
}
```

### `src/server.js`
```javascript
// Serves public folder as static files
app.use(express.static(path.join(projectRoot, 'public')));

// All routes default to index.html (SPA)
app.get('*', (req, res) => {
  res.sendFile(path.join(projectRoot, 'public', 'index.html'));
});
```
